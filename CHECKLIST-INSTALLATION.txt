  CHECKLIST D'INSTALLATION - OSMOSE ADS
═══════════════════════════════════════════════════════════════

AVANT L'INSTALLATION
───────────────────────────────────────────────────────────────
☐ Sauvegarde complète du site (base de données + fichiers)
☐ Vérifier la version WordPress (minimum 5.0)
☐ Vérifier la version PHP (minimum 7.4)
☐ Vérifier la mémoire PHP (recommandé 256MB)
☐ Activer le mode debug temporairement (wp-config.php)

INSTALLATION
───────────────────────────────────────────────────────────────
☐ Uploader le dossier 'osmose-ads' dans /wp-content/plugins/
☐ Vérifier que la structure est correcte
  └─ osmose-ads/
     └─ osmose-ads.php (fichier principal)
☐ Activer le plugin depuis Extensions
☐ Vérifier qu'il n'y a pas d'erreur fatale

CONFIGURATION INITIALE (OBLIGATOIRE)
───────────────────────────────────────────────────────────────
☐ Flush des permaliinks (Réglages > Permaliinks > Enregistrer)
☐ Configurer les réglages (Osmose ADS > Réglages)
  └─ Téléphone entreprise
  └─ Liste des services
☐ Ajouter au moins une ville (Osmose ADS > Villes)

TEST
───────────────────────────────────────────────────────────────
☐ Menu "Osmose ADS" visible dans l'admin
☐ Page Dashboard accessible
☐ Page Templates accessible
☐ Page Villes accessible
☐ Créer un template de test
☐ Créer une annonce de test
☐ Vérifier l'affichage public d'une annonce

SI PROBLÈME
───────────────────────────────────────────────────────────────
☐ Vérifier wp-content/debug.log
☐ Vérifier les permissions des fichiers (644/755)
☐ Désactiver les autres plugins temporairement
☐ Vérifier la version de WordPress et PHP
☐ Consulter GUIDE-INSTALLATION-SANS-ERREUR.md

═══════════════════════════════════════════════════════════════

IMPORTANT : Ne pas sauter l'étape "Flush des permaliinks" !
Elle est CRUCIALE pour que les URLs fonctionnent correctement.
