# Logo Osmose

Placez votre logo ici pour qu'il s'affiche automatiquement dans l'interface admin.

## Formats acceptés

- `logo.jpg`
- `logo.png`

## Emplacement

Le logo doit être placé dans : `/wp-content/plugins/osmose-ads/admin/img/logo.jpg` ou `.png`

## Taille recommandée

- Largeur : 200-400px
- Hauteur : proportionnelle
- Format : JPG ou PNG
- Fond : transparent de préférence

Le logo sera automatiquement détecté et affiché en blanc dans le header.
